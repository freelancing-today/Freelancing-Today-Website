# Freelancing Today Website

A lightweight, single-page marketing website for a freelance web service brand.

## Project Structure

- `index.html` – page structure and content sections
- `Styles.css` (capital `S`) – visual styling, layout, and responsive rules
- `script.js` – UI behavior (scroll nav state, reveal animations, counters, and contact form feedback)

## Run Locally

This project uses plain HTML, CSS, and JavaScript with no build step.

1. Open `index.html` directly in your browser, **or**
2. Serve the folder with any static server.

## Features

- Responsive layout for desktop and mobile breakpoints
- Fixed top navigation with scroll state
- Section reveal animations
- Animated hero counters
- Contact form UI feedback on submit

## Notes

- No package manager or test runner is configured in this repository.
- External dependencies are loaded via CDN in `index.html` (Bootstrap, jQuery, Popper, Google Fonts).
